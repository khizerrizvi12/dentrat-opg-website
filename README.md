# Dentrat Frontend

This is a static frontend starter for a Final Year Project website focused on dental OPG X-ray analysis.

## Files

- `index.html` - main website structure
- `styles.css` - responsive styling and layout
- `script.js` - upload preview and demo analysis interactions

## How to Run

Open `index.html` in any browser.

## What This Version Includes

- Landing page
- Feature overview
- Workflow section
- OPG upload demo
- Analysis result cards
- Dashboard snapshot
- Contact/footer
